How to load bulk/multiple trades or market data into FinPricing?


Trade

FinPricing provides user interfaces for uploading bulk trades. Bulk trade files are in comma-separated csv format. Users can load trades into Excel in the specified formats and then save them as a csv file.

FinPricing also provides user interfaces for uploading bulk market data. Again the data files are in comma-separated csv format specified here.

How to load bulk trades into FinPricing?

For example, a user wants to load multiple interest rate swaps into the system at once.

  1. Click the Tool tab at the top-left corner of the application. Next, expend Product -> InterestRateProduct. Then, select the Interest Rate Swap.


  2. Next, click the Load button. A window pops up allowing users to browse the local folders and select files.
  3. After selecting a bulk trade file that is in csv format, click the Open button.


  4. FinPricing starts to load all the trades defined in the file. Finally a summary table is displayed in the main window telling you how many trades are successfully loaded and how many of them fail.


Specifications and examples of bulk trade files

  • Fixed Income Products

How to load bulk market data into FinPricing?

For example, a user wants to load multiple interest rate curves into the system at once.

  1. Click the Tool tab at the top-left corner of the application. Next, expend Market -> InterestRateMarket. Then, select the Interest Rate Curve.


  2. Next, click the Load button. A window pops up allowing users to browse the local folders and select files.
  3. After selecting a bulk market data file that is in csv format, click the Open button.
  4. FinPricing starts to load all the market data defined in the file. Finally a summary table is displayed in the main window telling you how many data are successfully loaded and how many of them fail.


Specifications and examples of bulk market data files

  • Interest Rate Market
    • Interest Rate Curve
    • Bond Curve
    • Swaption Volatility Surface
    • Bond Market Quoted Price
  • Currency Market
    • FX Spot Rate
    • FX Forward Curve
    • FX Volatility Surface
  • Equity Market
    • Equity Spot Price
    • Equity Volatility
    • Equity Dividend
